返回格式：
    成功：
        info = {
            api: '接口名',
            code: 2000,     //成功状态码
            msg: '成功信息',
            flag: true,
            result          //处理结果，用户/产品信息
        }
    失败：
        info = {
            api: '接口名',
            code: 状态码,   //失败状态码
            msg: '错误信息',
            flag: false
        }

用户部分：
	http://10.3.141.65:2020/user
	
 	验证接口：
		get
		/uinfo/regcheck/:user_name
		请求参数：user_name 用户名 string
        返回参数：
            result: true/false
            
	注册接口：
		post
		/uinfo/reg
		请求参数：
            user_id     用户id      number
            user_name   用户名      string
            user_psw    用户密码    string
            user_token  用户token   string
        返回参数：
            result:true/false

	登录接口：
		post
		/uinfo/login
		请求参数：
            user_name   用户名      string
            user_psw    用户密码    string
        返回参数：
            result:true/false

    个人中心接口：
        get
        /uinfo/personal/:user_id
        请求参数：
            user_id     用户id      number
        返回参数：
            result:用户信息

    修改信息接口：（目前只有按id修改名字）
        put
        /uinfo/personal
        请求参数：
            用户所有信息
        返回参数：
            result:true/false

产品部分：
    http://10.3.141.65:2020/goods
    
    添加商品信息：
        post
        /goodinfo
        请求参数：
            {
                good_id,        商品id      number
                good_name,      商品名      string
                good_size,      款式        string
                good_price,     单价        number
                good_img,       图片        string
                good_stock,     库存        number
            }
            
        返回参数：
            result:true/false
            
    查询指定商品详情信息：
        get
        /goodinfo/:good_id
        请求参数：good_id   商品id      number

    查询所有商品详情信息：
        get
        /goodinfo
        请求参数：无
        返回参数：
            [
                {
                    good_id,        商品id      number
                    good_name,      商品名      string
                    good_size,      款式        string
                    good_price,     单价        number
                    good_img,       图片        string
                    good_stock,     库存        number
                }
            ]
            
购物车部分：
    http://10.3.141.65:2020/cart

    购物车添加商品:
        post
        /cartinfo
        请求参数：
            {
                user_id,        用户id      number
                good_id,        商品id      number
                good_name,      商品名      string
                good_size,      款式        string
                good_price,     单价        number
                good_img,       图片        string
                good_num,       商品数量    number
                good_stock,     库存        number
            }
        返回参数：
            result:true/false

    查询指定用户的购物车详情信息:
        get
        /cartinfo/:user_id
        请求参数：user_id       用户id      number
        返回参数：
            {
                user_id,        用户id      number
                good_id,        商品id      number
                good_name,      商品名      string
                good_size,      款式        string
                good_price,     单价        number
                good_img,       图片        string
                good_num,       商品数量    number
                good_stock,     库存        number
            }

    修改购物车商品数量:
        put
        /cartinfo
        请求参数：
            user_id         用户id      number
            good_id         商品id      number
            good_num        商品数量    number
